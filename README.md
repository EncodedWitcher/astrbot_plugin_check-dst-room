# 饥荒服务器大厅查询插件 - check-dst-room

一个用于 AstrBot 的插件，用于查询《Don't Starve Together（饥荒联机版）》服务器大厅信息

## 安装方式

我觉得你应该知道怎么装

## 可配置项

可以在插件管理界面的操作->配置插件中选择默认地区

可选地区包括：
- `ap-east-1`（默认）
- `us-east-1`
- `eu-central-1`
- `ap-southeast-1`

## 指令说明

使用 `/查房` 开启查询会话
